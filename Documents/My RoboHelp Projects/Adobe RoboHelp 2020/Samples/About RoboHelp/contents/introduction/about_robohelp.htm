About Adobe RoboHelp

The purpose of the About RoboHelp sample project is to highlight the features, explain how to use them in an easy to follow way and let you see some of them in use.
It is less detailed than Adobe's own online help but the aim is to get you going quickly. For many users that will be sufficient to get started, leaving the online help for the more detailed issues.
Use the project to test things without risking damage to your production projects.

Adobe RoboHelp 2019 shipped with a Classic version but also introduced a new version with a completely new UI whose features are described in this project. During the life of Adobe RoboHelp 2019 many updates added to the features and functionality.

These were further improved in Adobe RoboHelp 2020 which is licensed on a subscription only model. The subscription covers all updates and new versions.

This sample project will be updated from time to time but may not always be in sync with the latest product update. It is currently synced with Version 2020.5.

What's In Adobe RoboHelp?

The new and changed features in Adobe RoboHelp are highlighted in What's in Adobe RoboHelp and explained in the topics in this project.

Users upgrading from a Classic version, you should also see What's in Adobe RoboHelp 2019?

Upgrading New UI Versions

Upgrading refers to new versions and should not be confused with in-version updates. If you are upgrading to a higher version, before you start work note the Import Settings option on the Starter Screen. It allows you to import your settings into your new version.

Making Changes to the Sample Project

Should you make changes to the sample project that you cannot reverse, you can easily get a fresh copy. Delete the broken copy that is in Documents > My RoboHelp Projects and grab a fresh copy from Program Files. Don't attempt to open it from there, it will not function properly.

C:\Program Files\Adobe\Adobe RoboHelp 2020\resources\data\template\projects\About RoboHelp.

 With each update, both copies of the sample project are updated.
The copy in Program Files is replaced entirely.
In the copy in My RoboHelp Projects the files that are installed are also replaced but any files you have added will be left. If you want to retain your current copy with its installed files, rename the folder in My RoboHelp Project before you apply the update or move it to another location.
