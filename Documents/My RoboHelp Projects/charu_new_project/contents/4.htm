4. Formats in user manual

User manuals can be provided in either a paper format or as an electronic document (PDF or placed online or on-device in HTMTL). he user manual template is an MS Word document that can be printed or placed online.

User manuals can be created using a variety of tools. Each tool has its own advantages and disadvantages.
